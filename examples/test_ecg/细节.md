# 数据来源：

数据采用MIT-BIH Arrhythmia Database作为实验数据集，数据集中包含48条30分钟双导联动态心电记录，采样频率为360Hz，并附带人工标注的心搏事件。

# 主要数据处理：

1.正常心搏：`N、L、R、e、j`，编码为 0；

2.异常心搏：`A、a、J、S、V、E、F、/、f、Q`，编码为 1。

3.有效样本按8:1:1比例分为训练集、验证集和测试集，并采用分层抽样保证各子集类别比例与总体分布一致。

4.采用过采样平衡数据集，因为原始数据正常类占比90%。

# 模型结构：

输入张量形状为 `[batch_size, 1, 16, 16]`，网络结构如下：

1. `Conv2d(1, 4, kernel_size=3, stride=1, padding=1)`
2. `Conv2d(4, 8, kernel_size=3, stride=1, padding=1)`
3. `ReLU(inplace=False)`
4. `AdaptiveAvgPool2d((1,1))`
5. `Flatten(1)`
6. `Linear(8, 2)`
